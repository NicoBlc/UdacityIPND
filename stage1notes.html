History of the interwebs and how the technology works

So yeah...I went to wikipedia...turns out the world wide web has quite some history. Some guy named Tim Berners Lee invented the internet. In essence the world wide web is just a bunch of text files sitting on computer servers all linked together and approachable through Domain Name Servers (DNS). Where the domain name is sort of the home adress where you live. To visit an adress you need a webbrowser. Webbrowsers display client side code (HTML, CSS, JavaScript etc.). The code files stored on the server are... server side code (PHP, Python, Ruby etc.). Often ther are also databases involved (SQL or no-SQL).
